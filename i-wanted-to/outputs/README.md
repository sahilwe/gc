# GlobalConnectEducation NEET College Finder

Static React website for NEET aspirants to:

- Check India college options from NEET marks
- Move into an MBBS abroad funnel
- Filter abroad universities by country, city, university, and budget
- Compare agents by rating, experience, response time, admits, language, and counselling strength

## GitHub Pages

This folder is the site root. Publish this repository with GitHub Pages from the `main` branch and `/ (root)` folder.

The app uses React from the official CDN, so no build step is required.
